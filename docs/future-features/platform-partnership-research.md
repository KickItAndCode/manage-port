# Platform Partnership Research & Requirements

**Document Date:** June 18, 2025  
**Research Goal:** Prepare partnership applications for multi-platform listing syndication  
**Urgency:** HIGH - Applications can take 2-4 weeks for approval  

## Executive Summary

This research identifies key partnership opportunities for rental listing syndication across major platforms. Most platforms operate on partnership models rather than open API access, requiring formal applications and approval processes.

### Key Findings:
- **Apartments.com**: Partnership-based access through customer service representatives
- **Zillow Rental Network**: Free program requiring formal approval (2-4 weeks)
- **Syndication Services**: Landlord Studio and Rentec Direct offer existing multi-platform solutions
- **Timeline**: Most partnership approvals take 2-4 weeks minimum

## üè¢ Major Platform Partnership Requirements

### 1. Apartments.com API Partnership

**Partnership Model:** Customer/Vendor-based access only  
**Contact:** ocp-partnerships@costar.com  
**Application Process:** No public developer portal - requires direct contact  

#### Requirements:
- **Authentication:** OAuth 2.0 (Resource Owner Password Credentials Grant)
- **Technical Standards:** HTTP RPC-style methods via HTTPS
- **Access Type:** Granted to existing customers and approved vendors
- **Partnership Inquiry:** Must demonstrate mutual community clients

#### Application Preparation:
```
Business Information Needed:
- Company details and property management credentials
- Existing customer base size and geographic coverage
- Technical integration capabilities
- Mutual client references (if available)
```

**Estimated Timeline:** 3-4 weeks (vendor partnership track)

---

### 2. Zillow Rental Network Integration

**Partnership Model:** Free automated feed program  
**Contact:** Rentals Integrations team  
**Cost:** No cost to participate  

#### Requirements:
- **Pre-approval Required:** Must submit integration request before development
- **Target Users:** Property management companies and PM software providers
- **Feed Format:** XML conforming to Rental Listings Feed Guide or MITS format
- **Testing Phase:** Required before production access

#### Application Process:
1. Submit integration request to Rentals Integrations team
2. Receive approval and schedule developer consultation
3. Implement feed according to testing guidelines
4. Complete testing phase
5. Move to production syndication

#### Application Preparation:
```
Required Information:
- Company name and property management license information
- Number of properties and units managed
- Geographic markets served
- Technical contact information
- Integration timeline and development resources
```

**Estimated Timeline:** 2-3 weeks for approval + 3-4 weeks for implementation

---

### 3. RentSpree API Partnership

**Partnership Model:** Enterprise API integration  
**Contact:** Enterprise team - (323) 515-7757  
**Integration Timeline:** 2-3 weeks implementation  

#### Requirements:
- **Service Type:** Credit check API and rental application integration
- **Implementation:** Straightforward integration, minimal platform impact
- **Revenue Sharing:** Generates revenue for partners (documented $10M+ generated)
- **Platform Reach:** Serves 300+ MLSs and 2M+ users

#### Application Preparation:
```
Business Information Needed:
- Platform technical specifications
- Expected transaction volume
- Integration scope and timeline
- Revenue sharing expectations
```

**Estimated Timeline:** 1-2 weeks approval + 2-3 weeks implementation

## üîÑ Syndication Service Partnerships

### 1. Landlord Studio Syndication

**Service Type:** Built-in multi-platform syndication  
**Platforms:** Zillow, Trulia, Hotpads, Zumper  
**Cost:** Free syndication service  

#### Integration Approach:
- **API Integration:** Not required for users - handled internally
- **Developer Partnership:** Contact for white-label or API access
- **Processing Time:** 24 hours for listing approval and syndication

### 2. Rentec Direct Syndication Network

**Service Type:** Comprehensive syndication network  
**Contact:** partnerships@rentecdirect.com  
**Platforms:** Apartments.com, Zillow, Zumper, RentalSource, others  

#### Partnership Requirements:
- **Target Partners:** Outstanding services for landlords/property managers
- **Integration:** XML feed integration with existing Rentec network
- **Limitations:** 5 free Zumper listings/month, some platforms charge fees
